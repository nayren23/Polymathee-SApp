Projet APP RS1 – Analyse des Données d’Apprentissage

Un projet dirigé par M. Bougueroua visant à explorer les données de formation par apprentissage en Île-de-France.

Ce projet, réalisé en groupe de 4, exploite des données publiques sur l’apprentissage en créant une base de données relationnelle, des requêtes SQL adaptées et une interface de visualisation moderne.

Lien vers les données : data.gouv.fr – Apprentissage

  • Description détaillée du projet et des besoins
  • Modèle Entité-Association (MERISE)
  • Services et requêtes SQL correspondant à des problématiques métier
  • Visualisations de données (via PrimeNG Charts)
  • Interface utilisateur fonctionnelle en Angular / Flask
